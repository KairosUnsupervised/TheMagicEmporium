# The Magic Emporium — Project

## Project info
- **Plugin name:** The Magic Emporium
- **Platform:** FoundryVTT addon for D&D 5e (dnd5e system)

## Requirements
- **Extensive `.md` documentation.** Every system, feature, and API surface must be documented in Markdown. This includes user-facing guides (how to use the addon), GM-facing guides (how to author modifiers, templates, vouchers, wishes), developer-facing docs (data model, extension points), and content-pack authoring docs.
